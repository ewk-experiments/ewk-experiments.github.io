001
Chorus

Parallel multi-agent orchestration for VS Code.

Chorus decomposes a goal into a dependency-aware task graph and distributes it across autonomous worker agents. A Lead plans. Workers self-claim and execute. A Change Guardian arbitrates file conflicts across git worktrees. A QA Verifier blocks premature completion. The system runs until the goal is met or budget is exhausted.

002
Forge

Generative footwear. Text to 3D object.

Describe a shoe in natural language. Forge renders it as a multi-view image set via FLUX.2 and reconstructs a printable 3D mesh using Hunyuan3D-2.1. PBR materials. Arbitrary geometry. Every output is a unique specimen that has never existed before.

Image Model FLUX.2 Klein 4B
3D Pipeline Hunyuan3D-2.1
Output GLB / PBR mesh
Resolution 512-768 octree

003
Inlay

Inline completion for unstructured thought.

Inlay is a note-taking instrument that resolves incomplete information in real time. Mark unknowns with a single keystroke. Resolve all blanks with another. No chrome. No menus. No context switching. The interface is the text itself.
